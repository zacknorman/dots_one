DNS Lookup

Query the DNS server on all possible records. The default All option means resolving the A, AAAA, MX, NS and SOA records.

DNS Blackhole List

Check to see if an IP address is listed with nearly 100 DNS based anti-spam databases. Just type a domain name or an IP address and click the DNSBL Test button.

Domain Search

Find registered domain names based on one or more keywords. Keywords can be like a city, brand or trademark. Search through all Top Level Domains including gTLDs, nTLDs and ccTLDs.

Domain Typos

Find registered domain typos and misspellings for a given domain name. This tool shows all registered domains with one character difference of the given domain name. Search through all Top Level Domains including gTLDs, nTLDs and ccTLDs.

IP Geo Location

Show the geographical location for domain names or IP addressess on a map. IP addresses are assigned by international organisations like IANA and RIPE. Every assignment is public information and with other public available records it is possible to do a rough calculation of the geographical location of an IP address. Just type a domain name or IP address and click the Geo Location button.

Ping

Find out how long it takes for packets to reach its destination. Ping is from Germany. The network is from Hetzner. Just type a domain name or an IP address and click the Ping button.

Reverse IP

Find domains hosted on the same web server. Type a domain name or IP address and click the Reverse IP button.

Reverse MX

Find domains sharing the same mail server. Type a domain name or IP address and click the Reverse MX button.

Reverse NS

Find domains sharing the same name server. Type a domain name or IP address and click the Reverse NS button.

Reverse Adsense

Find domains sharing the same Adsense ID.

Reverse Analytics

Find domains sharing the same Analytics ID.

Traceroute

Traceroute is a tool to determine the route taken by packets across an IP network. Traceroute is from Germany. The network is from Hetzner.

Whois Lookup

Whois lookup is a powerful way to see all the registration information related to a particular domain, IP address or AS number. The information is stored into different databases. This lookup tool will query the right database and present the whois information in realtime. Just type a domain name, an IP address or an AS number and click the Whois button.
